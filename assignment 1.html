HTML Interview Questions and Answers

  1. What is HTML and what does it stand for?

    HTML stands for HyperText Markup Language. It is the standard language for creating web pages and web applications.

  2. What is the difference between HTML and XHTML?

    HTML is less strict than XHTML, which requires all elements to be properly nested and closed. XHTML is a reformulation of HTML as an XML application.

  3. What are the basic building blocks of an HTML document?

    The basic building blocks are elements such as <html>, <head>, <body>, and various tags for structuring content.

  4. How do you create a hyperlink in HTML?

    You can create a hyperlink using the <a> tag, like this: <a href="url">Link Text</a>.

  5. What is the purpose of the "alt" attribute in the HTML "img" tag?

    The "alt" attribute provides alternative text for an image if it cannot be displayed, improving accessibility and SEO.

  6. How can you create a table in HTML?

    You can create a table using the <table> tag along with <tr> for rows and <td> for data cells.

  7. What is the purpose of the "doctype" declaration in HTML?

    The "doctype" declaration informs the browser about the version of HTML being used, helping it render the page correctly.

  8. What is the difference between HTML elements and HTML tags?

    Tags are the markup used to define elements, while elements are the complete structure consisting of opening and closing tags and their content.

  9. How do you add comments in HTML?

    You can add comments using <!-- comment here -->.

  10. What is the significance of the "head" section in an HTML document?

    The "head" section contains metadata, links to stylesheets, scripts, and other information not displayed directly on the page.

  11. How do you define headings in HTML?

    Headings are defined using <h1> to <h6> tags, with <h1> being the highest level.

  12. What is the role of the "title" element in HTML?

    The "title" element defines the title of the document, which appears in the browser tab and is important for SEO.

  13. How do you create an ordered list in HTML?

    An ordered list is created using the <ol> tag, with each item inside <li> tags.

  14. What are the different types of form elements in HTML?

    Common form elements include text fields, radio buttons, checkboxes, dropdown menus, and submit buttons.

  15. How do you embed an image in an HTML document?

    You can embed an image using the <img src="url" alt="description"> tag.

  16. What is the purpose of the "href" attribute in an anchor tag?

    The "href" attribute specifies the URL of the page the link goes to.

  17. How do you create a line break in HTML?

    You can create a line break using the <br> tag.

  18. What is the purpose of the "target" attribute in an anchor tag?

    The "target" attribute specifies where to open the linked document, such as _blank for a new tab.

  19. What is the difference between block-level and inline elements in HTML?

    Block-level elements take up the full width available, while inline elements only take up as much width as necessary.

  20. How do you create a horizontal rule in HTML?

    A horizontal rule is created using the <hr> tag.

  21. What is the role of the "span" element in HTML?

    The "span" element is used for styling inline portions of text without affecting the layout.

  22. How do you create a checkbox in HTML?

    You can create a checkbox using the <input type="checkbox"> tag.

  23. What is the purpose of the "src" attribute in the "script" tag?

    The "src" attribute specifies the URL of an external script file to be included in the document.

  24. How do you create a dropdown menu in HTML?

    A dropdown menu can be created using the <select> tag, with options defined by <option> tags.

  25. What is the purpose of the "colspan" attribute in an HTML table?

    The "colspan" attribute allows a cell to span multiple columns in a table.

  26. How do you add a background image to an HTML element?

    You can add a background image using CSS, e.g., background-image: url('image.jpg');.

  27. What is the purpose of the "action" attribute in an HTML form?

    The "action" attribute specifies the URL to which the form data will be sent when submitted.

  28. How do you add a video to an HTML document?

    You can add a video using the <video> tag with controls for playback options.

  29. What is the purpose of the "disabled" attribute in an input field?

    The "disabled" attribute makes an input field non-interactive and unmodifiable.

  30. How do you create a form in HTML?

    A form is created using the <form> tag, enclosing input elements and a submit button.

  31. What is the purpose of the "placeholder" attribute in an input field?

    The "placeholder" attribute provides a hint to the user about what to enter in the field.

  32. How do you create a hyperlink with an image in HTML?

    You can create a hyperlink with an image using: <a href="url"><img src="image.jpg" alt="description"></a>.

  33. What is the purpose of the "required" attribute in an input field?

    The "required" attribute indicates that the input field must be filled out before submitting the form.

  34. How do you create a responsive layout in HTML?

    You can create a responsive layout using CSS media queries and flexible grid layouts or frameworks like Bootstrap.
